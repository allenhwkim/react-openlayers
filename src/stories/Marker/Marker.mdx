# {'<Marker />'}
 
A react representative of of `new ol.Feature(...)` customized as a marker  
https://openlayers.org/en/latest/apidoc/module-ol_Feature-Feature.html

## By Lon/Lat
<Story of={MarkerStories.Primary} />
<details open>
  <summary>Source code</summary>
  ```typescript
  import { OSM } from 'ol/source';
  import { Map, TileLayer, View, Marker } from 'react-openlayers';

  export default function(props) {
    return (
      <Map>
        <TileLayer source={new OSM()} />
        <View center={[0,0]} zoom={4}/>
        <Marker 
          lonLat={[0,0]} char="A" color="blue" 
          addOnClick={true} 
          removeOnClick={true} 
        />
        <PointerInteraction /> {/* Enable marker move */}
      </Map>
    );
  }
  ```
</details>

### props

| Prop   | type        | description |
| ------ | ----------- | ------------|
| lonLat        | number[] | Defaults to undefined. Longitude and latitude that marker to point, e.g. [0,0] | 
| address       | string   | Defaults to undefined. Address to show a marker, e.g. '1600 pennsylvania ave, Washington DC'. Internally it uses OSM nomination to get lon/lat.| 
| color         | string   | Defaults to 'red'. Fill color of the marker, e.g. 'red' |
| char          | char     | Defaults to ' '. A character for the marker. e.g. 'A' |
| addOnClick    | boolean  | Defaults to false. Add a new marker on click with a new character |
| removeOnClick | boolean  | Defaults to false. Remove the marker on click |
