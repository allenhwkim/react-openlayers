# {'<GraticuleLayer />'}
 
A react representative of of `new GraticuleLayer(...)`  
https://openlayers.org/en/latest/apidoc/module-ol_layer_Graticule-Graticule.html
<Story of={GraticuleLayerStories.Primary} />

### Usage
```typescript
import { Map, View, TileLayer, GraticuleLayer } from 'react-openlayers';
import Stroke from 'ol/style/Stroke';

export default function(props) {
  const strokeStyle = new Stroke({
    color: 'rgba(255,120,0,0.9)',
    width: 2,
    lineDash: [0.5, 4],
  })

  return <Map>
    <GraticuleLayer showLabels={true} strokeStyle={strokeStyle} />
    <View center={[-10997148, 4569099]} zoom={4}/>
  </Map>
}
```
