# {'<FullScreenControl />'}
 
A react representative of of `new FullScreen(...)`
https://openlayers.org/en/latest/apidoc/module-ol_control_FullScreen-FullScreen.html
<Story of={FullScreenControlStories.Primary} />

### Usage
```typescript
import OSM from 'ol/source/OSM';
import { Map, TileLayer, View, FullScreenControl } from 'react-openlayers';

export default function(props) {
  const mapRef = useRef();
  return (
    <Map ref={mapRef}>
      <FullScreenControl />
      <TileLayer source={new OSM()} label="F" />
      <View center={[-10997148, 4569099]} zoom={4}/>
    </Map>
  )
}
```

### props

You can pass any props defined in the 
[Openlayers documentation](https://openlayers.org/en/latest/apidoc/module-ol_control_FullScreen-FullScreen.html)
