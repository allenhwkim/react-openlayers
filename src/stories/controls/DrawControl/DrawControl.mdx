# {'<DrawControl />'}

Custom control to draw points, lines, polygons and circles. 
Press shift key for freehand drawing.

### How it works
1. Create a VectorLayer with a VectorSource, and applies to Map
2. Apply a Modify/Snap/Draw interaction with the VectorSource to Map

## How to use
1. Add `<DrawControl />`.
2. When click the toggle icon, you are ready draw. 
3. Pick a shape to draw. Press ESC to cancel draw.
5. Pick move icon to move the shape

