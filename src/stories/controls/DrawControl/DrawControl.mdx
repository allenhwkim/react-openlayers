# {'<DrawControl />'}

Custom control to draw points, lines, polygons and circles. 
Press shift key for freehand drawing.

### How it works
1. Create a VectorLayer with a VectorSource, and applies to Map
2. Apply a Modify/Snap/Draw interaction with the VectorSource to Map

## How to use
1. Add `<DrawControl />`.
2. When click draw icon, you are ready draw. 
3. Whend drawing done, click draw icon.

### How to draw
1. To cancel, press ESC key.
2. To move a shape, select move icon.
3. To delete a shape, select a shape then press delete  key.
5. For freehand drawing, press and hold shift key while drawing.
6. To save it, press shift-s to download geo json file.
